# 회원제 게시판 프로젝트(Spring Boot)



## ✔ 주요기능
___
***회원 관련 기능***
- 회원가입: 이메일(중복확인), 비밀번호, 이름, 전화번호, 프로필 사진
- 로그인
- 로그아웃
- 일반회원: 게시글 작성, 조회, 수정, 삭제
- 관리자: 회원목록조회, 회원삭제

***게시판 관련 기능***
- 글쓰기: 제목, 내용, 첨부파일, 작성자칸은 입력되어 있음
- 페이징 목록 출력: 한 화면에 5개씩글이 노출되고 페이지번호는 3개
- 글 수정, 삭제
- 작성자, 제목 검색
- 댓글 작성:  작성자칸은 입력되어 있음

***마이페이지 관련 기능***
- 회원정보 수정
- 프로필 사진 有
